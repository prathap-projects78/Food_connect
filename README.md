🍽Food Connect

Surplus Food Connect is a full-stack web application designed to minimize food waste by connecting restaurants with trusts/NGOs. The platform enables restaurants to list surplus food and allows trusts to claim it, with an admin-controlled approval workflow.

🚀 Features

✅ User Registration (Restaurant / Trust / Admin)
✅ Add Surplus Food Listings
✅ View Available Food
✅ Trust Accept Food Requests
✅ Admin Approve / Reject Requests
✅ Food Status Lifecycle (AVAILABLE → ACCEPTED → COLLECTED)
✅ DTO-based Clean API Responses
✅ Global Exception Handling
✅ MySQL Database Integration

🧠 Architecture

The system follows a layered architecture:

Controller Layer → REST APIs

Service Layer (optional if added)

Repository Layer → Database access

DTO Layer → Clean API responses

Model Layer → Entities

🔄 Workflow

1️⃣ Restaurant registers & logs food surplus
2️⃣ Food listing marked AVAILABLE
3️⃣ Trust accepts food → ACCEPTED
4️⃣ Admin approves → COLLECTED
